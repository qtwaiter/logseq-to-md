# Logseq to Markdown Converter

将 Logseq 的日记文件和 Markdown 文件转换为标准的 Markdown 格式。

## 功能说明

这个 Skill 专门用于处理 Logseq 特有的 Markdown 格式,将其转换为标准的、通用的 Markdown 格式。

### 主要功能

1. **移除 Logseq 特有语法**
   - 移除块前缀符号 (`-`)
   - 移除块属性 (`key:: value`)
   - 移除折叠状态标记 (`collapsed:: true`)
   - 保留查询块和渲染器(作为代码块显示)

2. **转换嵌套结构**
   - 将 Logseq 的缩进嵌套转换为 Markdown 标题层级
   - 保持内容层级关系

3. **保留所有内容**
   - 保留所有文本内容
   - 保留所有图片链接
   - 保留所有附件引用
   - 保留 Logseq 内部链接 (`[[...]]`)
   - 保留所有标签 (`#tag`)

4. **美化输出**
   - 添加适当的标题层级
   - 改善排版和可读性
   - 生成符合标准 Markdown 规范的文件

## 何时使用

**使用这个 Skill 当你需要:**
- 将 Logseq 日记分享给不使用 Logseq 的人
- 在其他 Markdown 编辑器中使用 Logseq 内容
- 将 Logseq 内容发布到博客或文档平台
- 归档或备份 Logseq 内容为标准格式
- 整理和规范化 Logseq 笔记

## 工作原理

### Logseq 格式示例
```markdown
- 一级标题
  - 二级内容
    - collapsed:: true
    - 三级内容
  - 另一个二级
    - 属性:: 值
    - #标签
```

### 转换后标准 Markdown
```markdown
# 一级标题

## 二级内容

### 三级内容

## 另一个二级

标签: #标签
属性: 属性:: 值
```

## 输出文件

- **默认保存位置**: `output/` 目录
- **文件命名**: 原文件名 + `_standard.md`
- **编码格式**: UTF-8
- **示例**: `2025_01_07.md` → `output/2025_01_07_standard.md`

## 支持的 Logseq 特性

| 特性 | 处理方式 |
|------|---------|
| Block 前缀 (`-`) | 移除,转换为标题层级 |
| 嵌套缩进 | 转换为 Markdown 标题 |
| 块属性 (`key:: value`) | 移除或转换为注释 |
| 折叠状态 (`collapsed:: true`) | 完全移除 |
| 图片引用 | 保留原样 |
| 附件链接 | 保留原样 |
| 内部链接 (`[[...]]`) | 保留原样 |
| 标签 (`#tag`) | 保留原样 |
| 查询块 (`{{query}}`) | 作为代码块显示 |
| 渲染器 (`{{renderer}}`) | 作为代码块显示 |

## 注意事项

1. **备份原文件**: 转换前建议备份原始 Logseq 文件
2. **路径问题**: 附件和图片路径保持不变,确保相对路径正确
3. **编码格式**: 使用 UTF-8 编码,避免中文乱码
4. **手动检查**: 转换后建议手动检查格式是否符合预期
